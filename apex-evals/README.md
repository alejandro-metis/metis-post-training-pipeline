# APEX Evals

Evaluation repository for Mercor's APEX benchmarks.

## Overview

This repository contains evaluation harness for evaluating large language models (LLMs) against the APEX benchmark suite.

## Getting Started

The main evaluation package for APEX-v1-extended is located in [`apex-evals-v1-extended/`](./apex-evals-v1-extended/). See that [documentation](./apex-evals-v1-extended/README.md) for:
- Installation instructions
- Usage examples
- API reference
- Supported models
